# StockPriceBot Static Demo

This folder contains a static GitHub Pages demo for StockPriceBot.

It does not require a backend, database, npm install, or build step. The page uses the screenshots from the repository `docs/` directory:

```text
demo/
  index.html
  styles.css
  app.js
```

## Preview Locally

Open `demo/index.html` in a browser.

## Publish on GitHub Pages

Recommended setup:

1. Commit the `demo/` and `docs/` folders.
2. Go to repository `Settings`.
3. Open `Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Choose branch `main` and folder `/ (root)`.
6. Open:

```text
https://<your-username>.github.io/<repository-name>/demo/
```

For this repository the URL will usually look like:

```text
https://<your-username>.github.io/StockPriceBot/demo/
```
