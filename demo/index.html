What the app covers

Core product-tracking workflows shown as a static portfolio demo.

P

Product tracking

Save name, URL, store, image, category, size, color, notes, price, and monitoring status.

W

Watchlist

Bookmark selected products and keep them in a dedicated fast-access view.

A

Alerts

Receive email alerts for price drops, target price, back-in-stock events, and variant availability.

C

Categories

Create categories with custom Lucide icons, colors, counters, and uncategorized product handling.

H

Health checks

Inspect monitoring status, selector issues, blocked stores, last successful checks, and logs.

E

Extension

Add products from a Chrome popup that reads page data and stores the API token locally.

S

Screen gallery

Switch between the main application areas.
